# Hier Blocks

A hierarchical block, or hier block, wraps multiple GNU Radio blocks into a
single reusable block. In GRC, creating a hier block lets a group of connected
blocks be represented and used as one block with defined parameters and input or
output ports.

Hier block documentation explains graph organization and reuse. It is not
mutation authority and does not authorize creating, deleting, or rewiring graph
content.
