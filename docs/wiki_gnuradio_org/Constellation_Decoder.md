# Constellation Decoder
From GNU Radio
[Jump to navigation](https://wiki.gnuradio.org/index.php?title=Constellation_Decoder#mw-head) [Jump to search](https://wiki.gnuradio.org/index.php?title=Constellation_Decoder#searchInput)
Decode a constellation's points from a complex space to (unpacked) bits based on the map of the object. See [Constellation Soft Decoder](https://wiki.gnuradio.org/index.php?title=Constellation_Soft_Decoder "Constellation Soft Decoder") for a soft output. 
## Parameters 

Constellation
    A constellation object, see [Constellation Object](https://wiki.gnuradio.org/index.php?title=Constellation_Object "Constellation Object")
## Example Flowgraph
See [Constellation Soft Decoder](https://wiki.gnuradio.org/index.php?title=Constellation_Soft_Decoder "Constellation Soft Decoder")
